### AI产品与功能更新

1.  **Google**为教育界推出了免费的**Gemini for Education** **AI助手** ✨📚，它基于**Gemini 2.5 Pro**和**LearnLM**模型，能为**教师**、**学生**及学校工作人员提供**教学计划**生成、**个性化教学**内容、**学习资料转播客**等功能，旨在提升学习和工作效率，并提供**数据安全保障**。只要学校使用**Google Workspace for Education**就能享受。
    <br/> ![Gemini教育AI助手界面](https://assets-v2.circle.so/q5appj1elksvfz3hbvruswj8d26r) <br/>
    <br/> ![Gemini教育版功能展示](https://assets-v2.circle.so/5v0gkf7hi4zgyuhkuvzphxhu7nxe) <br/>
    ['更多详情'](https://edu.google.com/ai/gemini-for-education/)

2.  **百度商业研发团队**推出了全球首个实现**中文音视频一体化生成**的创新模型"**MuseSteamer**”及其创作平台"**绘想**”！😮 这项技术在VBench I2V榜单上夺冠，仅需一张图片就能生成10秒**1080p电影级画质视频**，还能协同创作，有望**降低视频创作门槛**，**颠覆内容创作方式**，激发无限创意潜能！🎬✨
    <br/> ![百度MuseSteamer模型LOGO](https://pic.chinaz.com/picmap/202207140955455263_0.jpg) <br/>

3.  **微信**新推出的**AI搜索**功能引发了用户对**隐私**的担忧，因为它会自动将**姓名**转为**超链接**并生成详细个人信息，被网友戏称"强行开盒”😅。对此，**腾讯**回应说这只是整合了**公众号**和互联网的**公开信息**，并承诺会继续优化用户体验。🤔
    <br/> ![微信LOGO及界面](https://pic.chinaz.com/picmap/201911171903184300_9.jpg) <br/>

4.  **百度搜索**迎来了**十年来最大规模的改版**！🎉 它推出了支持**多模态输入**和**AI创作**的"**智能框**”、升级版"**百看**”，以及能视频通话的"**AI助手**”，旨在提供更智能、高效的用户体验。更棒的是，这次更新还集成了百度自研的全球首个**中文音视频一体化生成模型MuseSteamer**及其创作平台"**绘想**”，这标志着百度在**AIGC**领域取得了突破性进展！🥳
    <br/> ![百度搜索页面示意图](https://pic.chinaz.com/picmap/201912192146016232_0.jpg) <br/>

5.  **Google**旗下的**AI助手Gemini Live**迎来了**重大升级**！🎉 它将与**Google Maps**、**Google Calendar**、**Google Keep**、**Google Tasks**等应用**深度整合**，用户可以通过**语音或文字指令**实现**跨应用智能操作**，大幅提升效率和便利性。🤩 这项功能正逐步上线，未来还将连接更多Google应用，目标是成为一个兼顾**用户隐私**的**全能型智能助手**！
    <br/> ![Gemini Live界面示意图](https://upload.chinaz.com/2025/0702/6388706505118680904124074.png) <br/>

6.  武汉市汉阳区率先推出了全国首辆**AI外卖配送车**——"**智音车**”！🛵💨 这款车搭载了**北斗双频芯片**和全国首个即配专用号牌管理系统，试用者说效率提升了高达**30%**！由**北斗**和**雅迪**联合研发的"智音车”，凭借1米级的**定位精度**和多项智能技术，不仅提升了外卖配送效率，更为未来的**智能交通**奠定了基础，有望在全国推广！✨

7.  OpenRouter上线了一款名为「**Cypher Alpha**」的神秘**AI模型**，支持高达**100万token上下文**，而且**免费可用**！😮 它擅长**长上下文任务**和**代码生成**，主要用于收集用户反馈。这款模型的真实来源引发了**网友**对**OpenAI**可能进行**开源测试**的广泛猜测。虽然目前性能褒贬不一，但无疑给AI领域带来了一股神秘又引人关注的新风！👻
